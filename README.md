# Password Generator 🔑

An open source Flutter application to quickly generate strong and secure passwords.

## Features

- Instantly create complex passwords
- Choose the password length and which characters to include (letters, numbers, symbols, etc.)
- Simple, intuitive user interface
- Android support (iOS coming soon)
- One-tap copy to clipboard

## Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/password-generator-flutter.git
   cd password-generator-flutter
   ```

2. **Install dependencies:**
   ```sh
   flutter pub get
   ```

3. **Run the app:**
   ```sh
   flutter run
   ```

## Requirements

- [Flutter](https://flutter.dev/) SDK >= 3.0.0
- A terminal or editor compatible with Flutter (VSCode, Android Studio, etc.)
